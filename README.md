# Collectibles Market Research — Luxury Watch Index

Analysis of luxury watch market price appreciation benchmarked 
against traditional financial assets (S&P 500, Gold) over 2015–2025.

## Tools
Python · pandas · yfinance · matplotlib · openpyxl

## Key Findings
- Watch Index CAGR: 8.02% vs S&P 500: 8.71% (2015–2025)
- Watch market peaked at +218% in 2022 (post-COVID bubble)
- Post-peak correction of ~32% observed through 2025
- High volatility relative to S&P 500 consistent with illiquidity premium

## Files
- `collectibles_market_research.ipynb` — full analysis notebook
- `collectibles_market_dashboard.xlsx` — Excel dashboard with charts
- `watch_market_performance.png` — normalized performance chart
